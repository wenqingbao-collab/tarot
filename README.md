## 产品介绍

ARCANA 是一款融合塔罗占卜与 AI 深度解读的赛博风格占卜工具。

用户通过手势或鼠标从 78 张韦特塔罗牌中抽取三张，分别对应**过去、现在、未来**三个维度，AI 基于牌面组合与用户的问题生成专属解读报告，并支持下载 PDF 存档。

---

## 核心功能

- **沉浸式占卜体验** — 3D 星空场景 + 金色粒子特效，打造赛博神秘氛围
- **手势控制** — 开启摄像头后，用手势翻牌、选牌、接受命运（无需触碰屏幕）
- **鼠标模式** — 无摄像头也可正常使用
- **AI 深度解读** — 基于三牌组合与用户问题，生成约 800 字的个性化解读报告
- **PDF 一键下载** — 报告包含牌面图片、正逆位含义、AI 解读全文

---

## 使用流程

```
进入网站 → 启动占卜 → 抽取三张牌（过去/现在/未来）
    → 查看解读总结 → 填写问题 → 生成 AI 报告 → 下载 PDF
```

## 技术栈

| 层级 | 技术 |
|---|---|
| 前端 | 原生 HTML/CSS/JS + Three.js 3D 渲染 |
| 手势识别 | MediaPipe Hands |
| AI 解读 | DeepSeek API |
| PDF 生成 | jsPDF + html2canvas（客户端） |
| 部署 | Vercel（Serverless） |
| 域名 | Cloudflare |
